# Disclaimer

Rio Lab is an open-source project that provides scripts and guides for running local LLMs on personal hardware. This project is provided "as is" without warranty of any kind.

## What This Project Does

- Provides shell scripts to install and configure llama.cpp, OpenCode, and Open WebUI
- Downloads open-weight language models from HuggingFace
- Configures your local environment to use these tools

- Does **not** host or operate any cloud services
